# Plantilla Restaurante / Cafeteria · WebAlPaso

Landing page profesional para restaurantes, cafeterias, brunch, pastelerias y
locales gastronomicos. Construida con **React + Vite** y **CSS puro**.

## Caracteristicas

- Diseno editorial premium y 100% responsive
- Navbar transparente que se vuelve solida al hacer scroll
- Secciones: Hero, Nosotros, Carta, Galeria, Testimonios, Reserva, Ubicacion y Footer
- Formulario de reserva conectado a **Web3Forms**
- Boton flotante de WhatsApp
- Iconos con **Lucide React**
- Tipografias **Poppins** + **Fraunces**
- Listo para deploy en **Vercel**

## Personalizacion

Todo el contenido se edita en:

```txt
src/config/config.js
```

Ahi defines nombre del negocio, slogan, WhatsApp, email, direccion, horarios,
carta, galeria, testimonios, redes sociales, Google Maps y Web3Forms.

## Uso

```bash
npm install
npm run dev
npm run build
npm run preview
```
